# Editing Cards

Last modified: 17/03/2026

## Edit Paths

You can edit cards in multiple places:

- In the source note where the card was created
- In [Card Browser](./Card-Browser) for search, filter, and bulk updates
- During review sessions using the edit action

## Best Use

- Edit in the source note when changing meaning or structure.
- Edit in Card Browser when you need batch updates.
- Edit during review for quick corrections without leaving the session.

## Sync Note

After edits, run a sync so card metadata and schedule links remain aligned.
