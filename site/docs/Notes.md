# Notes

Last modified: 17/03/2026

## Notes First

LearnKit is designed for writing and learning directly from your markdown notes.

## Recommended Flow

1. Draft content in your note.
2. Add card blocks: [Card Formatting](./Card-Formatting)
3. Run sync: [Syncing](./Syncing)
4. Validate and edit cards: [Card Browser](./Card-Browser)

## Practical Rules

- Use clear headings and predictable structure for easier card extraction.
- Keep card prompts concise and answers specific.
- If parsing fails, check [Settings](./Settings) and quarantined card details.
