# Cards

Last modified: 13/02/2026

## Overview

Cards in Sprout are written inside your Markdown notes.
You can create cards by writing syntax directly or by using add-card forms.

## Card types

- [Basic & Reversed Cards](./Basic-&-Reversed-Cards.md): direct question/answer recall
- [Cloze Cards](./Cloze-Cards.md): fill-in-the-blank recall in context
- [Image Occlusion](./Image-Occlusion.md): hide parts of images
- [Multiple Choice Questions](./Multiple-Choice-Questions.md): choose the correct option
- [Ordered Questions](./Ordered-Questions.md): remember sequences

## Create cards

Two methods:

1. write card syntax in notes
2. use **Add flashcard** from the context menu or command palette

See [Creating Cards](./Creating-Cards.md), [Reading View](./Reading-View.md), [Reading View Styles](./Reading-View-Styles.md), and [Custom Reading Styles](./Custom-Reading-Styles.md).

## Edit cards

You can edit cards:

- in the source note
- in [Card Browser](./Card-Browser.md)
- during review (edit action)

See [Editing Cards](./Editing-Cards.md).

## Organize and format

- Use groups to organize cards.
- Use [Card Formatting](./Card-Formatting.md) for markdown, math, links, and images.
- Use [Flags](./Flags.md) for inline circle-flag tokens.

## Important behavior

- Broken or incomplete card syntax can cause cards to be skipped or parsed incorrectly.
- Card scheduling data is separate from note text (see [Backups](./Backups.md)).
