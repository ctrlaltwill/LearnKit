# Getting Started

Last modified: 17/03/2026

## First Run

1. Install LearnKit: [Installation](./Installation)
2. Run your first sync: [Syncing](./Syncing)
3. Create your first cards: [Flashcards](./Flashcards)
4. Start your first review: [Study Sessions](./Study-Sessions)

## Quick Model

- You author in notes.
- LearnKit parses cards.
- FSRS schedules reviews.
- You iterate from review feedback.

## Day One

1. Open [Cards](./Cards) and choose your first card type.
2. Add 10 to 20 cards from one note.
3. Run [Study Sessions](./Study-Sessions).
4. Review [Grading](./Grading) and [Scheduling](./Scheduling).
