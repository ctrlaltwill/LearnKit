# Tests

Last modified: 17/03/2026

## What Tests Does

Tests helps you run structured question rounds from your notes and card content.

## Typical Flow

1. Pick the scope (note, folder, or wider deck context).
2. Generate a test set.
3. Complete questions and review weak areas.
4. Convert weak points into targeted cards.

## Tips

- Use Tests for consolidation after study sessions.
- Keep tests topic-specific for better feedback loops.
- Pair with [Analytics](./Analytics) to track trends.
