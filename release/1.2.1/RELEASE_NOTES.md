### Release Date
2026-03-18

### Summary
LearnKit 1.2.1 is a stabilization release focused on polish, release quality, and compatibility fixes after 1.2.0.

## What Changed

### Build + Release
- Regenerated production artifacts in `dist/` for 1.2.1.
- Published matching release artifacts under `release/1.2.1/`.

### Quality + Compatibility
- Fixed Obsidian community plugin ESLint issues in affected async methods.
- Updated About copy and related settings styling improvements.
- Improved README messaging and release communication.

### Important Sync Note
- LearnKit scheduling storage now uses SQLite.
- For Obsidian Sync users, enable **Vault storage for scheduling databases** in LearnKit settings so scheduling data is synced from your vault.
